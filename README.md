
# 🚀 Web Lab Hello

Bu proje, React + TypeScript kullanılarak Vite ile oluşturulmuş basit bir web uygulamasıdır.

---

## 🛠️ Kullanılan Teknolojiler

- React
- TypeScript
- Vite
- Node.js
- npm

---

## 📦 Kurulum

Projeyi klonladıktan sonra bağımlılıkları yükleyin:

```bash
npm install

▶️ Geliştirme Sunucusunu Başlatma
npm run dev


Tarayıcıda açmak için:

http://localhost:5173

🏗️ Production Build
npm run build


Önizleme yapmak için:

npm run preview
